## Description
This repository houses my portfolio created with Node, React, and features Nodemailer. The 3 pages are about, portfolio, and a contact page. The contact page has a functional messaging system provided by NodeMailer that will send the content of the form directly to my email.

## Installation
Install using ```npm install```

## Usage
After installations, if you wish to run this locally, run the app using ```npm run dev```.
